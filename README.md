# 🍕 Pizza Bill Generator (Java Swing)

## 📌 Overview
Pizza Bill Generator is a simple Java Swing desktop application that helps users create and calculate pizza orders with a clean graphical interface. It allows customization of pizza type, base, and additional toppings, then generates a final bill.

---

## 🚀 Features
- Choose pizza base (Delux / Cheese / Toppings)
- Dynamic bill calculation
- Clean and simple GUI
- Instant bill generation

---

## 🛠️ Technologies Used
- Java
- Swing (JFrame, JButton, JRadioButton, JTextArea)

---

## ▶️ How to Run
1. Open the project in Eclipse or VS Code
2. Compile the Java file:
   ```bash
    javac PizzaUI.java
